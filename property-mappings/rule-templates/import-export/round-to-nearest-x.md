# Round to nearest X

Type: Import/Export

Scope: Supports only positive and negative integers.

<details>

<summary>Example</summary>

* Cell value: 1234.5678 mm
* Rule values:
  * Round to nearest X: 10
  * Ignore text: kg|Kg|g|mg|m|mm|each|L|ml|oz|fl
* Result: 1230

</details>
