# Calculate number

Type: Import/Export

Description: Uses the value of 's' ([rowData](../advanced-scripting.md)) and the equation to calculate and return a new number

<details>

<summary>Example</summary>

* Cell value: .07
* Rule value: n \* 100
* Result: 7

</details>
