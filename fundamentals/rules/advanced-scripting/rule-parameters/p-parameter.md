# p Parameter

The `p` parameter is the value of the `passOrBlock` value in a rule.&#x20;

This will evaluate to either `true` or `false`  depending on the setting selected in the UI for the rule.

#### Example&#x20;

To access the `p` parameter in a script, use&#x20;

```javascript
if (p === true)
{
  // do something
}
```
